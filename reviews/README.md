# 复盘库

复盘库的目的不是证明预测对错，而是区分：判断质量、价格质量、仓位纪律、结果噪音。

## 单场复盘模板

- Match:
- Events predicted:
- Bets placed:
- Final result:
- Profit/loss:
- Best read:
- Worst read:
- Was the price good:
- Was the stake right:
- Was the result noise:
- Lesson:

## 每日复盘模板

- Date:
- Total events tracked:
- Bets placed:
- Total stake:
- Net profit:
- Best event:
- Most ignored event:
- Best risk-reward event:
- Biggest mistake:
- Tomorrow bankroll adjustment:

## 周期复盘

每 3 天看一次：

- 哪类盘口表现最好。
- 哪类盘口亏损最多。
- 是否高估强队。
- 是否低估弱队受让。
- 是否过度参与高关注场。
- 是否错过低关注信息差。
