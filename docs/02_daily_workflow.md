# 每日工作流

## 早盘：比赛筛选

输出一张今日优先级表：

- A 档：必须研究，可能下注。
- B 档：观察为主，临场决定。
- C 档：只看结果，不下注。

筛选维度：

- 小组形势。
- 晋级动机。
- 阵容完整度。
- 强弱差距。
- 盘口是否有错价可能。
- 比赛时间是否适合临场跟踪。
- 是否有情绪热门队。

## 赛前 6-12 小时：初评

每场记录：

- 基础预测：胜平负、进球区间、节奏、风险。
- 关键变量：伤停、天气、裁判、轮换。
- 可下注事件：最多 1-3 个。
- 暂不下注原因。
- 预设入场价格。

## 赛前 90 分钟：首发确认

检查：

- 核心前锋是否首发。
- 核心中场是否首发。
- 中卫组合是否变化。
- 门将是否变化。
- 是否大面积轮换。
- 市场是否已经反应。

动作：

- 首发符合预案：允许按计划下注。
- 首发部分冲突：降级为观察或减半单位。
- 首发完全冲突：取消下注。

## 比赛中：临场触发

前 15 分钟只观察，不急着追：

- 是否高压。
- 是否主动提速。
- 射门质量是否真实。
- 角球是否来自持续压制。
- 黄牌是否改变防守动作。
- 裁判尺度是否偏严。

中场决策：

- 如果赛前逻辑成立但比分未体现，可考虑小额加仓。
- 如果赛前逻辑被比赛内容推翻，停止追加。
- 如果已经盈利，不因兴奋增加无关风险。

## 赛后复盘

每场必须归因：

- 预测正确，价格正确。
- 预测正确，入场价格差。
- 预测错误，信息遗漏。
- 预测错误，模型偏差。
- 结果噪音，过程合理。
- 情绪下注，纪律错误。

复盘不是为了证明自己对，而是为了找出下一场可以改进的变量。
