# 呈现效果设计

## 首页视图

首页只展示今天需要做的事：

- 今日比赛列表。
- A/B/C 优先级。
- 推荐研究事件。
- 可下注、等待、禁止下注三栏。
- 当日风险额度。
- 已使用仓位。
- 已确认首发数量。
- 临场触发提醒。

## 单场比赛页

每场比赛固定结构：

- 基础信息：时间、球队、组别、场地。
- 小组形势：积分、净胜球、出线概率、动机。
- 阵容：伤停、停赛、首发、轮换。
- 比赛剧本：快节奏、慢节奏、强队压制、弱队死守、互攻。
- 事件雷达：胜平负、让球、大小球、角球、牌、球员事件。
- 入场计划：价格、仓位、取消条件。
- 临场观察：15 分钟、30 分钟、中场、60 分钟。
- 赛后复盘。

## 每日简报格式

每日简报应控制在一屏内：

- 今天最值得看的 3 场。
- 今天最值得等待首发的 3 个事件。
- 今天不碰的高风险场次。
- 今日总仓位上限。
- 昨日复盘一句话。

## 仪表盘模块

后续可以做成网页或表格：

- 赛程日历。
- 小组积分模拟器。
- 晋级路径图。
- 事件雷达热力图。
- 仓位曲线。
- 命中率与收益率分解。
- 错误类型统计。

## 颜色与标签

- Green: 可下注。
- Yellow: 等待信息。
- Red: 禁止下注。
- Blue: 只观察。
- Gray: 已结束。

标签：

- `must-watch`
- `lineup-dependent`
- `motivation-edge`
- `public-bias`
- `in-play-only`
- `avoid`
- `review-needed`
