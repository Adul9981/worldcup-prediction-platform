# ADR-0002：网站 MVP 与商业方向

日期：2026-06-10。

## 决策

世界杯预测市场交易工具平台长期目标是订阅付费产品，但第一阶段先做本地只读 MVP，用来验证功能是否正常。

## 已确认产品选择

- 长期方向：订阅付费。
- 第一版：不做登录、不做账号、不做付费墙。
- 第一版：只读，不接钱包、不下单、不存私钥。
- Polymarket 市场跳转链接统一追加联盟参数：`?via=serene77mc-g6kj`。
- 首页结构采用：
  - 今日最值得关注事件。
  - 当前 Polymarket 预测市场选题状态。
  - 今日仓位 / 风险面板。
- 内容系统进入第一版。
- 评分采用双轨：
  - 稳健机会。
  - 高盈亏比观察。

## 原因

先验证数据、评分、仓位、复盘和内容输出是否可靠，再建设订阅商业化。这样避免过早建设账号、付费、权限等复杂系统。

## 后续影响

- 网站第一屏必须是实际工作台。
- 内容系统不是后置模块，而是 MVP 模块。
- 所有 Polymarket 外链需要统一处理联盟参数。
- 网站必须读取 `affiliate_url` 字段或统一链接生成函数，不能直接使用原始 `url` 渲染外链。
- 评分模型不能只有一个总分，必须同时输出稳健机会和高盈亏比观察。
